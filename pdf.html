1. Core Java & OOPs Principles
What is the difference between an Abstract Class and an Interface in Java?
Answer: An abstract class can have both abstract and concrete methods, instance state variables, and constructors. It supports single inheritance. An interface (prior to Java 8) could only have abstract methods; now it supports default and static methods. A class can implement multiple interfaces, making it ideal for defining peripheral capabilities decoupled from the main class hierarchy.
Explain the 4 Pillars of Object-Oriented Programming (OOPs).
Answer:
  • Encapsulation: Wrapping data (variables) and code (methods) together into a single unit and restricting direct access using private modifiers.
  • Inheritance: Mechanism where one class acquires the properties and behaviors of a parent class using the extends keyword.
  • Polymorphism: Ability of a function or object to take multiple forms. Compile-time is achieved via method overloading; Runtime is achieved via method overriding.
  • Abstraction: Hiding internal implementation complexities and showing only essential interfaces to the user.
How does Garbage Collection work in Java?
Answer: Java handles memory allocation automatically via the Garbage Collector (GC) running on the JVM. The GC targets the Heap memory area to track and identify objects that are no longer reachable by any live reference threads. Unreferenced objects are periodically swept and destroyed to free up system memory chunks.
2. Data Structures & Algorithms (DSA)
Compare an Array and a Linked List data structure.
Answer: Arrays store elements in contiguous memory locations, offering O(1) time complexity for random element lookups via indexing. However, its size is fixed at declaration. Linked Lists allocate nodes dynamically across arbitrary memory spaces connected via pointer references; lookups take O(N) sequential time, but insertions and deletions are highly efficient at O(1).
How do you detect a cycle in a Linked List structure?
Answer: Use Floyd's Cycle-Finding Algorithm (Two-pointer technique). Initialize a 'slow' pointer moving one node at a time and a 'fast' pointer moving two nodes at a time. If a cycle exists, the fast pointer will eventually wrap around and meet the slow pointer. If it hits null, no cycle exists.
3. Full-Stack Web Development & Databases
What is the difference between SQL and NoSQL databases? Compare MySQL and MongoDB.
Answer: MySQL is a Relational (SQL) database that stores structured data into predefined schemas, strict tables, rows, and relationships, utilizing ACID properties for transactional safety. MongoDB is a Non-Relational (NoSQL) document store mapping structured data inside dynamic JSON-like BSON schemas. MongoDB scales horizontally easily, whereas MySQL leans toward vertical scaling.
What is DOM Manipulation in JavaScript?
Answer: The Document Object Model (DOM) is an API programming interface mapping XML/HTML documents into standard tree objects. JavaScript uses DOM manipulation methods (like getElementById or querySelector) to dynamically access, alter, delete, modify structural properties, or style active UI nodes based on live browser actions.
4. IoT & Hardware Systems
What is the role of the ESP8266 microcontroller module in IoT architectures?
Answer: The ESP8266 is a low-cost, self-contained Wi-Fi microchip module featuring a built-in TCP/IP networking protocol stack. It allows standalone edge hardware controllers or Arduinos to hook directly into local WLAN interfaces, enabling direct edge-to-cloud server communication channels to pipe sensor readings.
